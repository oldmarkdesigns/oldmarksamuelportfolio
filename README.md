# Portfolio 2026

A modern portfolio website built with Next.js, React, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js app directory with pages and layouts
- `/components` - Reusable React components
- `/Portfolio Assets` - Image assets and resources
- `/public` - Static files (if needed)

## Pages

- `/` - Home page with hero, recent work, daily UI, and journey sections
- `/work` - All work projects
- `/work/[project]` - Individual project pages
- `/about` - About me page
- `/contact` - Contact page

## Build

To create a production build:

```bash
npm run build
npm start
```

## Technologies

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
